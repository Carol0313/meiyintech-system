------------------------------------------------------------
步骤3：执行一键部署脚本
------------------------------------------------------------

   chmod +x deploy/deploy.sh
   sudo bash deploy/deploy.sh

这个脚本会自动做以下事情（约10-20分钟）：
- 更新系统软件包
- 安装 Python3、Nginx、PostgreSQL
- 创建数据库和用户（自动生成随机密码）
- 创建Python虚拟环境并安装依赖
- 生成 .env 环境变量文件
- 执行 Django 数据库迁移
- 收集静态文件
- 配置Systemd服务
- 配置Nginx反向代理

执行过程中会问你：
"是否创建 Django 超级用户？(y/n)"
→ 输入 y，然后按提示创建（建议用手机号13800000000，密码admin123）

------------------------------------------------------------
常用运维命令（部署后使用）
------------------------------------------------------------

查看服务状态：
   sudo systemctl status magnesium

重启后端服务：
   sudo systemctl restart magnesium

重启Nginx：
   sudo systemctl reload nginx

查看错误日志：
   sudo tail -f /var/log/gunicorn/magnesium_error.log
